# US-03.1: Jornada de Ativação B2B (Empresários Alpha)

## 🎯 Objetivo (O que e por quê)
Criar uma régua de relacionamento automatizada e altamente persuasiva para leads que são empresários, utilizando dados demográficos e financeiros (Instagram, Faturamento, Localização) para quebrar a resistência inicial e converter em reuniões/vendas.

## 🧠 Contexto de Expert (Visão de Marketing/Produto estratégica)
Vender para quem fatura alto exige **especificidade**. Empresários ignoram automações "frias". Esta jornada usa o princípio de **Relevância Imediata**: citar o faturamento atual e o Instagram do lead sinaliza que você "fez o dever de casa". O tom deve ser direto, profissional e focado em resolver dores de escala proporcionais ao faturamento do lead.

## 📝 Fluxo de Experiência/UX (Passo a passo do usuário)
1. **Entrada**: Lead é importado ou criado com os campos `Faturamento` e `Instagram (@)`.
2. **Dia 0 (T+10min)**: Recebe uma mensagem de WhatsApp personalizada citando sua Cidade e Instagram.
3. **Dia 1 (T+24h)**: Follow-up sutil caso não tenha respondido, focando em uma dor comum da sua região.
4. **Dia 4 (T+96h)**: Entrega de valor (estudo de caso) específico para o nível de faturamento informado.
5. **Encerramento**: Se não houver resposta, o lead é marcado para "Tentativa Futura" para evitar saturar o contato.

## 📝 Requisitos Técnicos (Arquitetura, DB, Componentes)
- **Placeholders Dinâmicos**: O motor de envio deve suportar extrair dados de `custom_fields`.
- **Node de Espera**: Implementar lógica de `Wait` por horas/dias no motor de execução.
- **Personalização via IA**: (Opcional/Fase 2) Integrar com OpenAI para gerar o parágrafo de abertura baseado no nicho do Instagram.

## ✅ Critérios de Aceite (Checklist detalhado)
- [x] Mapeamento de Épico realizado.
- [x] Jornada criada no banco de dados com o ID do tenant correto.
- [x] Nodes de WhatsApp configurados com placeholders `{{custom_fields.Faturamento}}`.
- [x] Intervalos de tempo (Delays) configurados corretamente entre os passos.
- [ ] Landing page ou link de agendamento incluído na CTA final.
- [ ] Sistema de "Stop" se o lead responder (evitar continuar mandando spam).
